Choose Your Own Adventure (CYOA) Project

The CYOA project was to create an adventure where the user could make their own choices by the options that where given which resulted in different endings. In class we learned how to link markdown files and with this project we were able to practice those skills.

My partner, Amy Ponce, and I created a project "Going to School". Our topic was going to school so we created a plan on google drawings with some options like waking up or hitting snooze.

One challange me and my partner faced was linking files. We had many files in different folders which made it hard for the links to make it to the right destination. When Amy and I tried testing the links on github some links gave us an error message while some didn't. This was very confussing for us so we asked Mr. Mueller. The reason why our links didnt work was beause some of the files were in different folders which means we have to go into the folder and then the file. With Mr. Muellers help Amy and I were able to fix this problem which fixed our problem and made our links work.

Takeaways

    1. Communication and Collaboration is key 2. Don't be afraid to ask questions
